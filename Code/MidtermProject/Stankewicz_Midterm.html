HTML Table

HTML Cheat Sheet
Element name Element description Element example
Headings These tags are used to define HTML headings. h1 defines the most important heading and h6 defines the least important heading. <h1>
</h1>
Hyperlink The A tag defines a hyperlink. The most important attribute of the A element is the href attribute, which indicates the link's destination. <a>
</a>
Bold The B tag specifies bold text. <b>
</b>
Blockquote The BLOCKQUOTE tag specifies a section that is quoted from another source. Broswers usually indent the BLOCKQUOTE element. <blockquote>
</blockquote>
Line Break The BR tag inserts a single line break. The BR tag is an empty tag which means that it has no end tag. <br>
Caption The FIGCAPTION tag defines a caption for a FIGURE element. <figcaption>
</figcaption>
Figure The FIGURE tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc. <figure>
</figure>
Footer The FOOTER tag defines a footer for a document or section. <footer>
</footer>
Head The HEAD element is a container for all the head elements. The HEAD element can include a title for the document, scripts, styles, meta information, and more. <head>
</head>
Header The HEADER element represents a container for introductory content or a set of navigational links. A HEADER element typically contains ne or more heading element, logo or icon, authorship information. <header>
</header>
Horizontal Rule The HR tag defines a thematic break in an HTML page. The HR element is used to separate content or define a change in an HTML page. This element is an empty tag, which means there is no closing tag. <hr>
Main The MAIN tag specifies the main content of a document.The content inside the MAIN element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms. <main>
</main>
Paragraph The P tag defines a paragraph. <p>
</p>
Table The TABLE tag defines an HTML table. <table>
</table>
Table Head The THEAD tag is used to group header content in an HTML table. <table>
</thead>
Title The TITLE tag is required in all HTML documents and it defines the title of the document. The TITLE element defines a title in the browser toolbar, provides a title for the page when it is added to favorites, displays a title for the page in search-engine results <title>
</title>
